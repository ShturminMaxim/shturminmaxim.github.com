About me and my ideas.

Since 2008, my hobby has been to develop a web siteS based on various CMS. Using different templates, modules, plugins, components, it was managed to implement different ideas. But, unfortunately, I concluded that the use of ready functionality does not train me..

Since 2011, I have read the various documentation, have studied the specifications and methodologies for developing web applications. For a long time I was confined to theory.

In 2012, I began to develop seriously a web application from scratch, using PHP, MySql, JavaScript. Since then, AN interest in programming does not leave me for a minute.

In November, 2012 I took a three-month course "front-end development on JavaScript language" in "Maxymiser Academy". Every day I programme for 2-5 hours solving small tasks. I really want to learn more and work in a team with talented people.

I have my old online business card Koyot-Studio.com, continue to create sites based on different CMS.
