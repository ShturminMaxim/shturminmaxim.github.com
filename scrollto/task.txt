☐ Модуль навигации (по якорям)
   на странице расположены якоря. При подключении модуля навигации по якорям, генерируется меню из якорей. Имя, в которое сгенерируется пункт меню задается через data-title якоря. Если в якоре нет этого параметра, он не попадает в результирующее меню. Сгенерированне меню вставляется в узел, переданный в аргументе конструктору.

   new Navigation(nodeForMenu, onMenuSuccess);

   При клике на пункт меню с якорем происходит переход (scroll) к якорю на странице.

   Пункт меню, которому соотвествует якорь, до секции которого мы доскролировали, должен подсвечиваться активным.

   В момент, когда происходит скролл, подсвечивать проскроливаемые пункты меню нельзя.

   Можно давать ссылки на якоря из меню.
   Обработка двух одинаковых якорей

   Глобальные методы
     - getAnchors -> [names] (String)  +
     - goToAnchor ([name|index]) - переход к якорю, где name - значение якоря  +
   Модуль будет получать аргументом объект с методами (мадиатор)
     - publish
     - subscribe
   Модуль через медиатор паблишит события
     - 'navTo', аргументы коллбека — номер слайда и имя (происходит в момент перехода к слайду при клике на пункт меню)
     - 'navScrollTo', аргументы коллбека — номер слайда и его имя (происходит про скролле, не выстреливает, если скролл инициирован кликом по пункту меню в навигации)

     mediator.publish('navTo', {id:1, name: "about-them"});
